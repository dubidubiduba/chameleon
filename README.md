# `Chameleon`项目

## 项目简介
chameleon是一款桌面宠物插件，中文名译为”变色龙“，灵感来源于变色龙的变色特性，该宠物插件也能随着不同时间，不同环境，不同心情而选择自己喜欢的形象作为宠物主体，放置于桌面。

## 目录结构说明 
- documents -- 项目开发相关文档（如：项目需求文档、设计文档等等）  
- projects  -- 项目源码（比如：前端项目、后端项目等等）
  - characters --为宠物形象设计相关源码
  - images --资源素材等文件
  - windows--主窗体与各个子窗体源码
  - file --由程序引发的文件临时存储位置
